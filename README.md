# Front-end: Avaliação Serviço

Parte front-end do projeto de **Avaliação de Serviço**


Este projeto foi construído com:
- React
- Typescript
- Tailwind
- Vite


Para rodar o projeto basta utilizar os seguintes comandos:

```bash
npm i
npm run dev
```

E se certificar de ter o servidor da parte back-end rodando tranquilamente
